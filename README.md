### Récupérer le projet
Voici les étapes pour cloner un dépôt GitHub, de l'installation de Git (https://git-scm.com/) au téléchargement du projet :
- `Installer Git` : Téléchargez et installez Git depuis le site officiel.
- `Configurer Git` : Ouvrez votre terminal ou invite de commandes et configurez votre nom d'utilisateur et votre adresse e-mail Git :
```git config --global user.name "VOTRE NOM"```
git config --global user.email "VOTRE EMAIL"
- `Cloner le dépôt` : Sur GitHub, naviguez jusqu'à la page du dépôt, cliquez sur le bouton Code, et copiez l'URL (généralement HTTPS).
- `Exécuter la commande de clonage` : Dans votre terminal ou invite de commandes, naviguez jusqu'au répertoire où vous voulez placer le projet. Exécutez la commande git clone en utilisant l'URL que vous avez copiée :
git clone [URL du dépôt]
Le projet sera téléchargé dans un nouveau dossier portant le nom du dépôt.
